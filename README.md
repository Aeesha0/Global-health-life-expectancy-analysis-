# Global-health-life-expectancy-analysis-
An interactive Power BI dashboard analyzing global health and life expectancy using the World Bank World Development Indicators (2000–2024).

## Project Overview

This project was completed as the Week 8 Capstone Project for the AnalystLab Africa Data Analytics Internship.

The dashboard analyzes global health and life expectancy using the World Bank World Development Indicators (WDI) dataset. It focuses on four key indicators:

- Life Expectancy at Birth
- Maternal Mortality Ratio
- Infant Mortality Rate
- Current Health Expenditure (% of GDP)

Using Microsoft Power BI, the project explores global health trends, country comparisons, and the relationship between healthcare expenditure and health outcomes through interactive visualizations.

---

## Project Objectives

- Analyze global life expectancy trends.
- Compare countries with the highest and lowest life expectancy.
- Examine maternal and infant mortality.
- Explore the relationship between healthcare expenditure and life expectancy.
- Present insights through an interactive Power BI dashboard.
